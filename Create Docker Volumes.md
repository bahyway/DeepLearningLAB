To keep your data in a separate Docker container and import it when needed, you can use Docker volumes to share data between containers. This allows you to keep your data isolated in one container and access it from another container when necessary.

### Step-by-Step Guide

1. **Create a Data Container** :

* This container will be used to store your data.

1. **Create an Analysis Container** :

* This container will access the data from the data container and perform analysis.

1. **Use Docker Volumes** :

* Docker volumes can be shared between containers to allow the analysis container to access data stored in the data container.
