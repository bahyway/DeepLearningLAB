### Explanation

1. **Logging Driver** :

* The `logging` key allows you to configure log drivers and options for each service.
* `driver: "json-file"` specifies that Docker should use the `json-file` logging driver, which writes logs in JSON format.

1. **Logging Options** :

* `max-size`: Specifies the maximum size of the log file before it is rotated. In this example, it's set to `200k` (kilobytes).
* `max-file`: Specifies the maximum number of log files to keep. In this example, it's set to `10`.
